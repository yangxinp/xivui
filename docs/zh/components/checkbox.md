## Checkbox API

### 属性

| 属性名        | 说明           | 类型                  | 默认值  | 是否必填 |
| ------------- | -------------- | --------------------- | ------- | -------- |
| value         | 绑定值         | Boolean/String/Number | `false` | No       |
| color         | 颜色           | String                | `blue`  | No       |
| indeterminate | 设置不确定状态 | Boolean               | `false` | No       |
