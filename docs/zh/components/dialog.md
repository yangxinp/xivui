## Dialog API

### 属性

| 属性名     | 说明                      | 类型    | 默认值  | 是否必填 |
| ---------- | ------------------------- | ------- | ------- | -------- |
| value      | 是否显示                  | Boolean | `false` | No       |
| title      | 标题                      | Boolean | `false` | No       |
| modal      | 显示遮罩                  | Boolean | `false` | No       |
| width      | 对话框的最大宽度          | String  | -       | No       |
| persistent | 点击外部和按下esc不会关闭 | Boolean | `false` | No       |
