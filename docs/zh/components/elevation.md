# Elevation 海拔

和 [Color 颜色](./color.md) 一样，关于阴影方面也内置了样式类，方便开发者快速改变元素的海拔。在元素上的 `class` 属性加入 `elevation-${number}` 即可。 `number` 范围是 `0 ~ 24`，海拔随着数字的增加而增高。

:::
