# Icon 图标

该组件库内置 Material Design 图标，可以通过 [Material Design Icons](https://pictogrammers.com/library/mdi/icon) 来查看所有图标。讲类型传入 `x-icon` 的 `type` 属性即可。

:::demo icon/basic

:::

## Icon API

### 属性

| 属性名 | 说明     | 类型   | 默认值 | 是否必填 |
| ------ | -------- | ------ | ------ | -------- |
| type   | 图标类型 | String | -      | No       |
